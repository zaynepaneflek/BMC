# BMC Facility Landing Page

A polished single-page website for BMC Facility, an elite baseball training center based in Aruba.

## What is included

- `index.html` — page structure and content
- `styles.css` — responsive visual design and layout
- `script.js` — scroll reveal animation, carousel controls, and mobile navigation toggle
- `Images/` — local image assets used by the page

## How to view

1. Open `index.html` in your browser.
2. For best results, host locally using a simple HTTP server if you need the Google Maps iframe to load correctly.

## Notes

- The page includes a responsive hero section, coach and pricing sections, testimonials, and contact details.
- Mobile navigation is supported with a menu toggle on smaller screens.
- The WhatsApp booking button opens a new chat window.
